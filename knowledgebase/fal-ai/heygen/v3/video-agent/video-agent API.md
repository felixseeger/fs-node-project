## Overview

- **Endpoint**: `https://fal.run/fal-ai/heygen/v3/video-agent`
- **Model ID**: `fal-ai/heygen/v3/video-agent`
- **Category**: text-to-video
- **Kind**: inference


## Pricing

For more details, see [fal.ai pricing](https://fal.ai/pricing).

## Usage Examples

### cURL

```bash
curl --request POST \
  --url https://fal.run/fal-ai/heygen/v3/video-agent \
  --header "Authorization: Key $FAL_KEY" \
  --header "Content-Type: application/json" \
  --data '{
     "prompt": "Create a 30-second product demo video showcasing our new AI features with professional lighting and modern graphics."
   }'
```

### Python

Ensure you have the Python client installed:

```bash
pip install fal-client
```

Then use the API client to make requests:

```python
import fal_client

def on_queue_update(update):
    if isinstance(update, fal_client.InProgress):
        for log in update.logs:
           print(log["message"])

result = fal_client.subscribe(
    "fal-ai/heygen/v3/video-agent",
    arguments={
        "prompt": "Create a 30-second product demo video showcasing our new AI features with professional lighting and modern graphics."
    },
    with_logs=True,
    on_queue_update=on_queue_update,
)
print(result)
```

### JavaScript

Ensure you have the JavaScript client installed:

```bash
npm install --save @fal-ai/client
```

Then use the API client to make requests:

```javascript
import { fal } from "@fal-ai/client";

const result = await fal.subscribe("fal-ai/heygen/v3/video-agent", {
  input: {
    prompt: "Create a 30-second product demo video showcasing our new AI features with professional lighting and modern graphics."
  },
  logs: true,
  onQueueUpdate: (update) => {
    if (update.status === "IN_PROGRESS") {
      update.logs.map((log) => log.message).forEach(console.log);
    }
  },
});
console.log(result.data);
console.log(result.requestId);
```
