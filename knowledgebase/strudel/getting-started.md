# Getting Started - Strudel

## Welcome
Strudel is the official port of the **Tidal Cycles** pattern language to JavaScript. It allows you to expressively write dynamic music pieces using code. You do not need to know JavaScript or Tidal Cycles to start; the interactive tutorial is designed to guide you through the basics.

## What can you do with Strudel?
*   **Live code music:** Create music with code in real-time.
*   **Algorithmic composition:** Use Tidal's unique approach to pattern manipulation.
*   **Teaching:** Its low barrier of entry makes it ideal for teaching music and code simultaneously.
*   **Integration:** Use it as a flexible sequencer via MIDI or OSC.
