# Firestore Security Rules Tests

## Prerequisites
- Node.js 18+
- Java JRE installed (for Firebase Emulator)

## Running the Tests
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the Firebase Emulator in one terminal:
   ```bash
   npm run emulator:start
   ```
3. Run the tests in another terminal:
   ```bash
   npm test
   ```
